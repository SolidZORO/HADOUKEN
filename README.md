dayone2html
===========

處理dayone for mac的文件，並生成html。



使用方式
------------
git回來， 在當前目錄 node run.js 一下就可以生成index索引和日誌。
目前版本需要做一些手工修改，比如圖片需要自己拷貝到attachment/origin下，tag合集功能也沒有，還不能一鍵，但總算成型了。
